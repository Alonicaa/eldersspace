Promo Code Manager (จัดการโค้ดส่วนลด)

อัพโหลดโค้ดสำหรับแคมเปญ
ขั้นที่ 1: เลือกรางวัล *
โค้ดที่อัพโหลดจะถูกผูกกับรางวัลที่เลือก จำนวนโค้ดในระบบจะอัพเดทอัตโนมัติจากไฟล์ที่อัพโหลด
ประวัติการอัพโหลดโค้ดสำหรับรางวัลนี้
ขั้นที่ 2: อัพโหลดไฟล์โค้ด (CSV)

อัพโหลด CSV — ง่าย รวดเร็ว รองรับโค้ดจำนวนมาก

Format ไฟล์ CSV:
Header row: code,description,expiry_date
ตัวอย่าง: PROMO001,ส่วนลด 50 บาท,2026-12-31
* description และ expiry_date ไม่บังคับ
